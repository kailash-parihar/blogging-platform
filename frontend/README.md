
# Frontend

This directory contains the frontend code for the blogging application.

## Technologies Used

- React
- Vite
## Getting Started

### Installation

1. Install dependencies:
   ```markdown
   npm install
   npm run dev
   ```
   
## Available Scripts
   
   **npm run dev:** 
   - Start the development server
   
   **npm run build:** 
   - Build the project for production
   
   **npm run serve:** 
   - Serve the production build
